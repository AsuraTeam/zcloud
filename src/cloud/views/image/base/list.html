上传基础镜像
1. 在本地 docker 环境中输入以下命令进行登录,使用管理员账号登录
docker login reg1.zcloud.com:49000

2. 然后，对本地需要 push 的 image 进行标记，比如：
docker tag java8:latest reg1.zcloud.com:49000/template/java8:latest

3. 最后在命令行输入如下命令，就可以 push 这个 image 到镜像仓库中了
docker push reg1.zcloud.com:49000/template/java8:latest

需要在镜像中安装docker服务,并且安装基础环境,比如java8, maven,git等工具
请注意替换仓库域名
